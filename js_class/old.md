## 核心提炼

在 javaScript 中，类的实现是基于其原型继承机制的。注意这里的继承一词是不准确的，应该称之为“原型引用”。因为一个 js 类实例只是拥有原型上所有属性的引用，而不是一个副本。
javaScript 这门语言是动态的，具有非常高的灵活性。所有 js 类的一个重要特性就是“动态可继承”
